# Telegraph Floooder

## Requirements

- Python 3.x
- Telegraph
- Colorama

## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/goidaa/telegraph-flooder.git
    ```

2. Install the required modules:

    ```bash
    pip install telegraph
    pip install colorama
    ```

## Usage

```bash
cd telegraph-flooder
python main.py
```
